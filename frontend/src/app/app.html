Plantilla full stack

Angular + Spring Boot

El frontend esta preparado para consumir una API en /api. Define primero el contrato REST y usa los agentes del proyecto para construir la primera funcionalidad.

Orden recomendado

  1. @api-contract define rutas y JSON.
  2. @database @backend implementa Spring Boot.
  3. @front-connector crea pantallas y servicios Angular.
  4. @testing @deploy verifica y construye.
