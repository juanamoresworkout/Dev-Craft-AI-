# DTO

Esta carpeta contendra objetos para las peticiones y respuestas JSON de la API.

Ejemplos:

- Datos para crear o actualizar un recurso.
- Respuestas combinadas, como un resumen con datos de varias entidades.
- Mensajes de validacion o error acordados en el contrato.

Los DTO permiten que el contrato REST no dependa directamente de las tablas.
